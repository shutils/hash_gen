# hash_gen

## 実行環境

このアプリケーションは以下の環境で動作することを想定しています

- OS:Windows
- Python:3.9

また、pipenv がインストールされている必要があります

## 仮想環境起動

```
$ pipenv shell
$ pipenv sync
```

## 起動（開発時）

プロジェクトのルートディレクトリで以下のコマンドを実行するとホットリロード状態で開発を行うことができます。

```
$ flet run app.py
```

## ビルド

以下のコマンドで実行形式ファイルを作成することができます

```
$ flet pack app.py
```
